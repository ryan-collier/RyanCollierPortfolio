# Ryan Collier — Portfolio

Personal portfolio site for Ryan Collier — MSc Data Science student at
Lancaster University and BSc (Hons) Mathematics graduate from the
University of Bath.

**Live site:** https://ryan-collier.github.io/RyanCollierPortfolio

## About

A single-page site covering:

- **About** — background and academic focus
- **Skills** — languages, frameworks, and tools used
- **Projects** — write-ups of data science work
- **Contact** — email and GitHub

## Built with

- HTML5
- CSS3 (custom properties, Grid/Flexbox, no framework)
- Vanilla JavaScript (scroll-reveal via IntersectionObserver)

No build step — open `index.html` directly, or serve locally:

```bash
python -m http.server 5500
```
